Lab 1 – Build an Alexa Skill using Alicebot

  • Submit your report before 11:59pm on April ?, 2019

The goal of this lab is to get to acquainted with the Alexa Skills Kit and ngrok. We will go through the workflow of hosting an Alicebot on your laptop as an Alexa Skill endpoint. At the end of this lab, you should be able to talk with the Alicebot through an Alexa-enabled device.

Preparation

  1. Create an Amazon Developer account.
  2. Create an ngrok account.
  3. Follow instructions in this Github repository.

Optional: AWS Lambda

AWS Lambda is recommended by Amazon for Alexa Skill endpoints to handle requests at scale. For this course, you don't have to use an AWS Lambda endpoint. If you are interested, you can follow the instruction below to use AWS Lambda to replace the your local server and get rid of the ngrok.

  1. Sign up for Amazon AWS. Note that you should to sign up with your @uw.edu account. You will get Free Tier access.
  2. (Optional) Sign up for Amazon Educate. You will need your account ID which you can find under "my account" from the AWS Admin Panel. This should give you a $100 credit for using AWS. [FAQ]
  3. Follow instructions in this Github repository.
